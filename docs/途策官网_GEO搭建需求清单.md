## 一、首页必须有的GEO元素

### 1. 结构化数据（Schema标记）— 最重要的技术项

在首页HTML的`<head>`里加入这段JSON-LD，让AI一抓取就知道你是谁：

```json
{
  "@context": "https://schema.org",
  "@type": "EducationalOrganization",
  "name": "途策必达教育科技",
  "alternateName": "途策留学",
  "description": "一家以策略驱动的留学申请机构，专注帮助学生用最优路径实现海外名校梦想",
  "url": "https://你的官网.com",
  "logo": "https://你的官网.com/logo.png",
  "foundingDate": "2023",
  "address": {
    "@type": "PostalAddress",
    "addressLocality": "上海",
    "addressRegion": "上海",
    "addressCountry": "CN"
  },
  "areaServed": "全国",
  "serviceType": ["美国留学申请", "英国留学申请", "留学规划", "文书指导", "面试辅导"],
  "knowsAbout": ["美本申请", "英本申请", "研究生申请", "留学规划"],
  "parentOrganization": {
    "@type": "Organization",
    "name": "上海途策必达教育科技有限公司"
  }
}
```

### 2. 首页文案要点

```
标题区（H1）：途策留学 — 让每一个申请都有策略
副标题：专注海外TOP50院校申请 | 策略定制·拒绝模板

3句话简介（放在显眼位置）：
① 途策留学成立于上海，专注高端海外升学规划
② 3对1专属团队：策略顾问+文书导师+流程管家
③ 申请季前6-12个月启动个性化策略制定
```

## 二、必须有的页面（按优先级）

### P0：关于我们 ⭐⭐⭐

直接使用我上次写的GEO友好公司简介。附加内容：
- 创始人照片+简介+行业经验年限
- 团队合照/办公环境照片（实体感）
- 公司成立时间、服务学生总数（可验证数据）

### P0：成功案例页 ⭐⭐⭐

**这是你最大的GEO杠杆——AI最喜欢引用真实案例。**

每个案例的结构：
```
学生姓名（可化名）：XXX
录取院校：XX大学（专业排名）
标化成绩：托福XXX SAT/ACT XXX
背景亮点：XXX活动/竞赛
申请策略：一句话概括
学生感言：1-2句
```

**SEO/GEO标题格式：**
```
星河湾 → 耶鲁大学 | 途策留学2026录取案例
平和 → 哈佛大学语言学 | 途策留学学员故事
```

**技术要点：** 每个案例页加上 `ItemList` 和 `Review` Schema标记。

### P1：服务页面

| 页面 | 内容 |
|:-----|:------|
| 美国本科申请 | 服务流程、时间线、团队、价格 |
| 美国研究生申请 | 同上 |
| 英国本科/研究生申请 | 同上 |
| 背景提升 | 活动规划、竞赛辅导、夏校申请 |
| 文书服务 | 文书创作流程、导师介绍、范文（脱敏） |

### P1：团队介绍

每个顾问单独一页，包含：
- 照片
- 姓名+头衔
- 从业年限
- 擅长领域
- 成功案例数
- 教育背景
- 个人签名/理念

**这是E-E-A-T的关键——AI会抓取"人"的信息来判断可信度。**

### P1：FAQ页面

**AI最喜欢FAQPage Schema，直接提高被引用率。**

```
Q: 途策留学和别的机构有什么不同？
A: 我们的核心差异是"策略先行"……

Q: 途策留学的服务流程是怎样的？
A: 第一步：免费评估 → 第二步：制定策略 → 第三步：执行申请 → 第四步：跟进结果

Q: 途策留学的收费是多少？
A: 根据申请目标和服务范围不同，费用从X万到Y万不等……

Q: 签约后如果对顾问不满意可以换吗？
A: 可以。我们在签约合同中明确了顾问更换机制……

Q: 途策留学主要做哪些国家的申请？
A: 美国、英国、加拿大、澳大利亚……
```

**技术实现：** 在FAQ页面加入 `FAQPage` Schema。

### P2：博客/干货文章区

每月至少4篇GEO优化文章，主题方向：
- 选机构类（决策词）→ "上海留学中介怎么选不踩坑"
- 趋势类（认知词）→ "2026年美本申请趋势分析"
- 场景类（场景词）→ "托福100分能申什么学校"
- 对比类（决策词）→ "大机构vs小工作室哪个更适合你"

## 四、技术清单（给开发人员）

```html
<!-- 每个页面必须包含 -->
1. <title> 格式：核心关键词 | 途策留学
   例："上海留学中介哪家好 | 途策留学"

2. <meta description> 格式：包含核心关键词+一句话价值
   例："上海留学中介怎么选？途策留学专注海外TOP50院校申请，策略定制不套模板，2026申请季开放咨询。"
   
3. JSON-LD Schema（见第一部分）

4. Open Graph 标签（微信分享时显示）
   <meta property="og:title" content="..."/>
   <meta property="og:description" content="..."/>
   <meta property="og:image" content="..."/>

5. 中文SEO基础
   - 每个页面有唯一的H1
   - URL使用拼音（如 /meiben-shenqing）
   - 内链丰富（文章互相链接）
   - 网站地图 sitemap.xml
```

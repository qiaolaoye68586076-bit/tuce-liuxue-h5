# 设计素材清单

以下是当前页面缺失或建议补齐的视觉素材，方便直接发送给设计师。

## 1. 品牌与 logo
- 主 logo：`logo.webp`（96×96，当前已使用；由原 logo-dark.svg 内嵌位图压缩而来）
- 可选：logo 在深/浅背景下的备用版本
- 可选：品牌徽标 badge/图标组件素材

## 2. 首屏 Hero
- 1 张高清首屏背景图（`assets/hero-bg.jpg`，必须）
- 可选：Hero 叠加图形装饰或品牌氛围插画

## 3. 导师头像
- 4 张真实导师肖像图（当前已使用占位图）
  - 规划导师
  - 文书导师
  - 个性化导师
  - 学术导师

## 4. 学员案例图
- 3 张真实学员/录取案例图（当前 `assets/case-01.jpg`, `assets/case-02.jpg`, `assets/case-03.jpg`）
- 建议图像风格：案例氛围照、校园场景、录取喜悦类图

## 5. 二维码图
- 顾问微信二维码：`assets/qr-code.png`
- 官方公众号二维码：`assets/qr-official.png`

## 6. 关键视觉补强（建议补图）
这些区块目前是文字+图标形式，建议补一套品牌风格插图或背景图：
- 服务模块（美国本科 / 研究生 / 转学 / 英欧加港本科 / 单项服务 / 商务合作）
- 本科服务体系 / 三大模块
- 为什么选途策（品牌优势）
- 途策方法论（流程与体系）
- 名校文书训练营
- Offer 墙（可选：院校校徽 / 精选图标）

## 7. 内容与授权说明
- 学员故事图必须是真实授权素材，替换当前占位图
- 导师头像必须真实、专业、统一风格
- 用户协议与隐私政策内容需要正式文案

## 8. 建议提供的设计规范
- 色彩方案（深绿 / 暖金 / 米白 / 黑白）
- 字体规范（标题、正文字体、英文/中文搭配）
- 图标风格说明（线性图标、实心图标、装饰纹理）
- 图片裁剪比例与尺寸规范
